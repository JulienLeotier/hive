# v0.3 Release Notes

v0.3 lands external framework adapters, the HiveHub registry, a NATS backend,
and vector-based knowledge search.

## New features

### Adapter dispatch on `hive add-agent --type`

| Type         | Transport             | Required flags                 |
|--------------|-----------------------|--------------------------------|
| `http`       | HTTP                  | `--url`                        |
| `claude-code`| local / HTTP          | `--url` or `--path`            |
| `mcp`        | HTTP                  | `--url`                        |
| `crewai`     | subprocess            | `--path`                       |
| `autogen`    | subprocess            | `--path`                       |
| `langchain`  | subprocess            | `--path`                       |
| `openai`     | OpenAI Assistants API | `--assistant-id` + `--api-key` |

Auto-detection based on project markers (AGENT.yaml, CLAUDE.md, crewai.yaml,
`.mcp.json`, `autogen_agent.py`, `langchain_agent.py`) kicks in when `--path`
is supplied without an explicit `--type`.

### HiveHub — template registry

```bash
hive search code-review                # search the public HiveHub index
hive install <template> --dest <dir>   # materialise a template locally
hive publish . --name my-hive --version 0.1.0 --description "…"
```

Templates are JSON manifests (`[{ path, content }, …]`) so the publish flow
produces a file you can attach to a PR against the HiveHub index.

### NATS event backend

`event.EventBus` is now an interface. Two implementations ship:

- `event.Bus` — SQLite-backed (default; durable, single-node).
- `event.NATSBus` — for multi-process deployments. Publishes to
  `hive.events.<type>`, fan-outs to local subscribers, and keeps a bounded
  in-memory history so `Query()` still works without JetStream. Wire it via
  any `NATSConn` implementation (satisfied by `*nats.Conn`).

### Vector knowledge search

`knowledge.Store` gains an `Embedder` interface + two built-ins:

- `NewHashingEmbedder(dim)` — zero-dependency deterministic baseline.
- `NewOpenAIEmbedder(apiKey, model, fallback)` — hits `/v1/embeddings`; falls
  back to the provided embedder on any error (rate limit, 5xx, timeout, no key).

Auto-populates the `embedding` BLOB on `Record`, and adds `VectorSearch(query, limit)`
that ranks entries by cosine similarity.

### Cost & budget features (backport)

- `hive status --costs` — per-agent spend rollup + breach flags.
- `hive budget set <agent> <daily-limit>` + `hive budget list|remove`.
- `/api/v1/costs` endpoint and the `/costs` dashboard page.

## Migrations

- `004_task_checkpoint_timestamp.sql` — adds `tasks.checkpoint_at`.
- `005_trust_overrides.sql` — per-task-type trust level overrides.

## Breaking changes

None. v0.2 databases upgrade in place.
